#  Financial Personality Classification (NLP + Deep Learning)

##  Project Description
The **Financial Personality Classification** project uses **Natural Language Processing (NLP)** and **Deep Learning** to identify an individual's **financial personality type** based on their investment habits, preferences, and textual inputs.  
It classifies users as **Safe Investor**, **Balanced Planner**, or **Risk Taker** by analyzing both their written responses and numerical investment data.

This project can be useful for **financial advisors, fintech platforms**, and **banks** to understand user behavior and provide personalized financial recommendations.

---

##  Objective
To build a hybrid AI model that combines **text data** and **numeric financial features** to predict the type of investor a person is, using advanced **machine learning** and **deep learning** techniques.
